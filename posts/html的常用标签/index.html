HTML的常用标签

August 18, 2019

a 标签的用法

  1. 属性
  • href
  • target
  • download
  • rel=noopener

a 的 href 取值

  1. 跳转到外部页面

    <a href="//baidu.com">百度</a>

  1. 跳转到内部锚点

    <p id="xxx">10</p>
<a href="#xxx">链接到xxx</a>

点击 链接到 xxx 就到了 10 号

  1. 跳转到电话或者邮件

    <a href="mailto:1054804846@qq.com">发邮件给黄维</a>
<a href="tel:13700951618">打电话给维维</a>

在手机上打开该网页点击就可以直接呼出邮件 app 或者呼出拨号键了（并且电话号码会自动输入进去）

a 的 target 取值

  1. 内置名字
  • _blank 在空白页面打开
  • _top 顶级页面打开
  • _parent 在上一级页面打开
  • _self 默认值，在自己页面打开
  1. 程序员命名

  • window 的 name 做一个页面导航

    <a href="//google.com" target="xxx">google</a>
<a href="//baidu.com" target="xxx">baidu</a>

这样做在打开一个页面后（假如是 google），那么再点击 baidu 按钮，就会覆盖刚刚打开的那个页面，变为 baidu.com，因为这两个命令都要用一个叫做 xxx 的 window，所以会互相覆盖

  • iframe 的 name 做一个 goodbai 网站

    <a href="//google.com" target="xxx">google</a>
<a href="//baidu.com" target="xxx">baidu</a>
<iframe src="name=xxx" ></iframe>

这样可以实现在同一个页面同时用百度和谷歌查资料（不过谷歌已经把 iframe 这个功能在 google 网站上禁用了）

image 标签的用法

<!DOCTYPE html>
<html lang="zh-CN">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Document</title>
    <style>
      * {
        margin: 0;
        padding: 0;
        box-sizing: border-box;
      }

      img {
        max-width: 100%;
      }
    </style>
  </head>

  <body>
    <img id="xxx" src="812superfast.png" alt="一台法拉利" />
    <script>
      xxx.onload = function() {
        console.log("图片加载成功");
      };
      xxx.onerror = function() {
        console.log("图片加载失败");
        xxx.src = "/404.png";
      };
    </script>
  </body>
</html>

  1. 作用 发出 get 请求，展示一张图片

    <img id="xxx" src="812superfast.png" alt="一台法拉利" />

  1. 属性

  • alt

    alt="一台法拉利"

当图片加载出现错误时，出现一个图片的名称，至少可以减少用户想弄死我的冲动

  • scr 以及事件 onload/onerror 或者还有个更好的选择，当法拉利出不来的时候，也就是事件 onerror 发生时，那就出来一个奥拓代表加载失败了吧，至少奥拓也是车啊,看了有木有舒服点

    <img id="xxx" src="812superfastx.png" alt="一台法拉利" />
<script>
xxx.onload = function() {
console.log("图片加载成功");
};
xxx.onerror = function() {
console.log("图片加载失败");
xxx.src = "/404.png";
};
</script>

  • 响应式 max-height/max-width 图片宽度以及高度

    <style>
* {
margin: 0;
padding: 0;
box-sizing: border-box;
}
img {
max-width: 100%;
}
</style>

在 rest 一些参数后，在给一个

max-width: 100%;

或者

max-height: 100%

这样就可以做到例如图片宽度或者高度会自适应当前窗口大小，与之相对的另外一个宽高参数会随着比例自动改变

table 标签的用法

  1. 相关的标签
  • table
  • thead 头
  • tbody 身体
  • tfoot 脚部
  • tr table row 一行
  • td table date 数据

  • th 表头

    <!DOCTYPE html>
<html lang="zh-CN">
<head>
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<meta http-equiv="X-UA-Compatible" content="ie=edge" />
<title>Document</title>
<style>
  table {
    width: 300px;
    table-layout: auto;
    border-collapse: collapse;
    border-spacing: 0;
  }
  td,
  th {
    border: 1px solid blue;
  }
</style>
</head>
<body>
<table>
  <thead>
    <tr>
      <th></th>
      <th>小红</th>
      <th>小明</th>
      <th>小颖</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th>数学</th>
      <td>61</td>
      <td>91</td>
      <td>85</td>
    </tr>
    <tr>
      <th>语文</th>
      <td>79</td>
      <td>82</td>
      <td>92</td>
    </tr>
    <tr>
      <th>英语</th>
      <td>100</td>
      <td>97</td>
      <td>87</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <th>总分</th>
      <td>200</td>
      <td>200</td>
      <td>200</td>
    </tr>
  </tfoot>
</table>
</body>
</html>

  1. 相关的样式

  • table-layout 属性定义了用于布局表格单元格，行和列的算法。
    auto
    意思就是每一列和行的宽度是自动的 （大多数浏览器采用自动表格布局算法对表格布局，表格及单元格的宽度取决于其包含的内容）
    fixed
    意思就是等宽（表格和列的宽度通过表格的宽度来设置，某一列的宽度仅由该列首行的单元格决定。在当前列中，该单元格所在行之后的行并不会影响整个列宽。）

  • border-collapse
    border-collapse CSS 属性是用来决定表格的边框是分开的还是合并的。在分隔模式下，相邻的单元格都拥有独立的边框。在合并模式下，相邻单元格共享边框。
