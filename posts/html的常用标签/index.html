HTML的常用标签

August 18, 2019

a 标签的用法

  1. 属性
  • href
  • target
  • download
  • rel=noopener

a 的 href 取值

  1. 跳转到外部页面

    <a href="//baidu.com">百度</a>

  1. 跳转到内部锚点

    <p id="xxx">10</p>
<a href="#xxx">链接到xxx</a>

点击 链接到 xxx 就到了 10 号

  1. 跳转到电话或者邮件

    <a href="mailto:1054804846@qq.com">发邮件给黄维</a>
<a href="tel:13700951618">打电话给维维</a>

在手机上打开该网页点击就可以直接呼出邮件 app 或者呼出拨号键了（并且电话号码会自动输入进去）

a 的 target 取值

  1. 内置名字
  • _blank 在空白页面打开
  • _top 顶级页面打开
  • _parent 在上一级页面打开
  • _self 默认值，在自己页面打开
  1. 程序员命名

  • window 的 name 做一个页面导航

    <a href="//google.com" target="xxx">google</a>
<a href="//baidu.com" target="xxx">baidu</a>

这样做在打开一个页面后（假如是 google），那么再点击 baidu 按钮，就会覆盖刚刚打开的那个页面，变为 baidu.com，因为这两个命令都要用一个叫做 xxx 的 window，所以会互相覆盖

  • iframe 的 name 做一个 goodbai 网站

    <a href="//google.com" target="xxx">google</a>
<a href="//baidu.com" target="xxx">baidu</a>
<iframe src="name=xxx" ></iframe>

这样可以实现在同一个页面同时用百度和谷歌查资料（不过谷歌已经把 iframe 这个功能在 google 网站上禁用了）

image 标签的用法

<!DOCTYPE html>
<html lang="zh-CN">

<head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <meta http-equiv="X-UA-Compatible" content="ie=edge" />
    <title>Document</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
        }

        img {
            max-width: 100%;
        }
    </style>
</head>

<body>
    <img id="xxx" src="812superfast.png" alt="一台法拉利" />
    <script>
        xxx.onload = function () {
            console.log("图片加载成功");
        };
        xxx.onerror = function () {
            console.log("图片加载失败");
            xxx.src = "/404.png";
        };
    </script>
</body>

</html>

  1. 作用 发出 get 请求，展示一张图片

    <img id="xxx" src="812superfast.png" alt="一台法拉利" />

  1. 属性

  • alt

    alt="一台法拉利"

当图片加载出现错误时，出现一个图片的名称，至少可以减少用户打电话让经理开出我的冲动 或者还有个更好的选择，当法拉利出不来的时候，那就出来一个奥拓代表加载失败了吧，至少奥拓也是车啊,看了有木有舒服点
